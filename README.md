# Fürstenwalde Produktpage

Statische Produktseite für `fuerstenwalde.riccardopopp.de`.

## Struktur

- `index.html` - Seiteninhalt und SEO-Basics
- `styles.css` - responsives Layout und visuelles Design
- `app.js` - Header- und Reveal-Verhalten
- `assets/` - App-Icon, Stadtmotiv und App-Screenshots

Zum lokalen Prüfen kann `index.html` direkt im Browser geöffnet werden.
